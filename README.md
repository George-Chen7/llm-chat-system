# 🗣️ 基于LLM的智能对话系统

一个支持文字与语音双向交互的智能对话平台，采用Vue 3 + Go前后端分离架构，集成大语言模型API，提供高性能的多模态交互体验。

## ✨ 核心特性

### 💬 智能对话
- 基于大语言模型的上下文感知对话
- 实时流式消息响应，支持逐字输出效果
- 多轮对话上下文管理，智能记忆压缩

### 🎤 多模态交互
- **文字交互**：Markdown渲染、代码高亮、数学公式
- **语音交互**：实时语音识别(STT) + 文字转语音(TTS)
- **文件支持**：PDF、Word、图片等多格式文档解析

### 👥 用户系统
- 完整的用户注册/登录系统
- 对话历史记录与多会话管理
- 智能配额控制与使用统计
